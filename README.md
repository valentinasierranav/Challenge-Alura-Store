# Challenge-Alura-Store
# 📊 Análisis Comparativo de Plataformas de Venta (Tiendas 1–4)

## 🎯 Objetivo del Proyecto
Este proyecto presenta un análisis comparativo de datos de cuatro plataformas de venta independientes: **Tienda 1, Tienda 2, Tienda 3 y Tienda 4**.

El objetivo principal es identificar cuál de las plataformas resulta menos conveniente para continuar con la inversión, con el fin de apoyar la toma de decisiones estratégicas basadas en datos.

Para ello, se analizan diferentes dimensiones del desempeño de cada tienda, incluyendo:

- **Métricas financieras**, como ingresos y rentabilidad.
- **Indicadores operacionales**, relacionados con el volumen y comportamiento de las ventas.
- **Aspectos de calidad del servicio**, como evaluaciones o patrones de satisfacción del cliente.

A partir de este análisis, se busca **minimizar el riesgo de inversión y maximizar la utilidad**, proporcionando una base objetiva para determinar en qué plataforma conviene continuar operando.
