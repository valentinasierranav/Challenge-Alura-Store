# 📊 Análisis Comparativo de Plataformas de Venta

## 📌 Descripción del Proyecto

Este proyecto presenta un análisis comparativo de datos entre cuatro plataformas de venta independientes (Tienda 1, Tienda 2, Tienda 3 y Tienda 4).  
El objetivo principal es evaluar el desempeño de cada tienda utilizando diferentes métricas de negocio para apoyar una decisión estratégica de inversión.

A través de técnicas de análisis exploratorio de datos y visualización, se busca identificar cuál de las tiendas presenta menor potencial de crecimiento y podría ser considerada para desinversión.

---

## 🎯 Objetivo

Determinar, mediante el análisis de datos, cuál de las cuatro tiendas representa la opción menos favorable para continuar la inversión, permitiendo optimizar recursos y concentrar esfuerzos en las plataformas con mayor potencial de rentabilidad.

---

## 📂 Datos Analizados

El conjunto de datos incluye información de ventas de cuatro tiendas diferentes, con variables como:

- Producto  
- Categoría del producto  
- Precio  
- Costo de envío  
- Método de pago  
- Lugar de compra  
- Vendedor  
- Calificación del cliente  
- Fecha de compra  

---

## 🔎 Análisis Realizados

Durante el proyecto se llevaron a cabo distintos análisis exploratorios y comparativos:

### 1️⃣ Análisis de Facturación
Se calcularon los ingresos totales generados por cada tienda para evaluar su desempeño financiero general.

### 2️⃣ Ventas por Categoría
Se analizaron las categorías de productos más vendidas para comprender los patrones de demanda de los clientes.

### 3️⃣ Calificación Promedio de Clientes
Se evaluó el nivel de satisfacción de los clientes mediante la calificación promedio de los productos vendidos.

### 4️⃣ Productos Más y Menos Vendidos
Se identificaron los productos con mayor y menor volumen de ventas para detectar tendencias de consumo.

### 5️⃣ Relación entre Ventas e Ingresos
Se analizó si un mayor volumen de ventas implica necesariamente una mayor facturación.

### 6️⃣ Costo Promedio de Envío
Se calculó el costo promedio de envío por tienda para evaluar posibles diferencias en competitividad logística.

---

## 📊 Visualización de Datos

Para facilitar la interpretación de los resultados, se desarrollaron visualizaciones utilizando **Matplotlib** y **Seaborn**, incluyendo:

- Gráfico de barras de ingresos totales por tienda
- Gráfico de distribución del ingreso total
- Comparación del costo promedio de envío
- Gráfico de burbujas para analizar ventas por categoría

Estas visualizaciones permiten identificar patrones y diferencias entre las tiendas de forma más clara.

---

## 🧠 Conclusiones

El análisis realizado muestra que, aunque las cuatro tiendas presentan niveles de facturación relativamente similares, existen diferencias importantes en su desempeño.

La **Tienda 4** presenta el menor rendimiento financiero y una menor participación en categorías de productos de alto valor, lo que sugiere un mercado con menor potencial de crecimiento.

Aunque esta tienda presenta un costo de envío promedio más bajo, esta ventaja no es suficiente para compensar su menor nivel de ventas y facturación.

Por esta razón, se recomienda que el **Sr. Juan considere vender o cerrar la Tienda 4**, reasignando los recursos disponibles hacia las tiendas con mejor desempeño, especialmente la **Tienda 2**, donde el capital invertido podría generar mayores retornos y oportunidades de crecimiento sostenido.
